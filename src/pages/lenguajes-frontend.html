Lenguajes del Frontend

Los paquetes de módulos son herramientas que los desarrolladores frontales utilizan para agrupar módulos JavaScript en un solo archivo JavaScript que se puede ejecutar en el navegador.

Alguno de los module bundlers son: webpack, parcel, babel, rollup, fusebox, etc.

HTML

Empaquetador de aplicaciones web, que se diferencia por la experiencia ofrecida a los desarrolladores.
